# 项目矩阵

这个目录用于沉淀可放进简历的项目，而不是零散 Demo。

## P1：React 数据看板

目的：训练 React 组件、状态、请求和图表能力。

建议功能：

- 数据列表
- 搜索、筛选、排序
- 图表展示
- 加载状态和错误状态
- README 解释组件结构和状态设计

## P2：Next.js 全栈 CRUD

目的：训练前端页面、Node API、数据库和部署闭环。

建议功能：

- 登录
- 列表、新增、编辑、删除
- 表单校验
- 错误处理
- 部署上线

## P3：WebGIS + Agent 空间数据分析平台

目的：作为主项目，体现 GIS + 前端 + Agent 的差异化。

最小闭环：

```text
上传 GeoJSON
  -> 地图展示
  -> 属性筛选
  -> 空间统计
  -> 图表联动
  -> Agent 生成分析摘要
```

推荐技术栈：

- React / Next.js / TypeScript
- Mapbox GL JS 或 Leaflet
- Turf.js
- Node API
- Python 数据处理脚本
- OpenAI API 或 Codex 辅助工作流

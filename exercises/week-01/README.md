## 代码要求

- 每个练习单独建一个 `.js` 或 `.ts` 文件。
- 文件顶部写 3 行说明：功能、输入、输出。
- 至少补 2 个边界情况。
- 写完后让 Codex 做一次严格 review。

## 验收标准

你需要能回答：

```text
1. 这个函数为什么需要闭包？
2. 变量什么时候创建，什么时候释放？
3. 如果输入为空、null、重复值，会发生什么？
4. 这段代码的时间复杂度和空间复杂度是多少？
```

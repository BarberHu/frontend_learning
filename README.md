# React/Next + WebGIS + Agent 前端就业路线

这是一个面向前端就业的 6 个月学习与作品集仓库。

目标不是泛泛地“学完整个计算机科学”，而是先打穿一个可求职的工程闭环：

```text
JavaScript / TypeScript
  -> React / Next.js
  -> Node 全栈闭环
  -> WebGIS 空间数据分析平台
  -> Agent 智能查询与自动分析
  -> 简历、面试、项目深挖
```

## 求职定位

目标标签：

```text
React/Next/TypeScript 前端工程师
  + Node 能做全栈闭环
  + Python/GIS 能做数据处理
  + Agent 能做智能化交互与自动分析
```

核心差异化：

- 不做同质化 Todo、商城、后台模板项目。
- 主项目围绕 WebGIS、空间数据、地图交互和 Agent 分析能力展开。
- 简历不写“熟悉 AI”，而写具体能力：自然语言空间查询、地图筛选解释、空间分析摘要生成。

## 使用规则

- 每周只做 1 个主知识点、1 个项目功能、1 组算法题。
- 每周至少留下一个可见产物：代码、Demo、README、截图或复盘。
- Codex 只作为解释器、评审员、测试员和面试官，不作为无脑代写工具。
- 如果开始反复比较课程但没有产出，立刻回到本周日志，做一个 30 分钟微任务。
